# APFC — Automatic Power Factor Correction

APFC is Peepul Farm's IoT-enabled power factor correction device, designed for agricultural pump sets and small industrial loads. It monitors power quality and automatically switches capacitor banks to maintain optimal power factor.

## Key Components

| Component | Description |
|-----------|-------------|
| APFC Controller | Embedded controller with power metering and relay switching |
| Cloud Relay | 4G/WiFi module for remote monitoring via DemeterCloud |
| Admin Portal | Web dashboard for fleet-level energy monitoring |
