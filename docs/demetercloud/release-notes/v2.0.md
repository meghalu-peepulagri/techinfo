# DemeterCloud v2.0 — 2026-04-18

**Platform:** Web ([app.demetercloud.com](https://app.demetercloud.com))

---

## Overview

This release delivers an upgraded mission planning pipeline with a new C++ path generation algorithm, full obstacle support, improved error visibility, manual path generation mode, and mission cancellation.

---

## What's New

### Updated Path Planning Algorithm

Upgraded the backend path planning service to the new C++ build (`metis_addon.node` — `generateMission`). The algorithm now consumes the full `mission_planner_input` format directly, including field boundary, origin coordinates, coverage settings, and obstacles. Deployed on Railway.

### Planner Error Handling

Errors returned by the C++ algorithm (e.g. `"No coverage region found in the map"`, code 21) are now propagated to the user with a clear message. Previously these caused silent failures or server crashes. The API now returns a structured `422` response with the planner's error description.

### Obstacle Drawing on Field Map

Users can draw obstacles directly on the field map when configuring a mission:

- **Circle** — for round obstacles (ponds, trees, poles)
- **Rectangle** — for rectangular obstacles (boulders, structures, sheds)

Drawn obstacles can be selected from the sidebar list, moved, resized using map handles, renamed inline, and removed via the ✕ button or the Delete key. All obstacles are included in the mission payload sent to the planning algorithm so the generated path avoids them.

### Manual Mode for Path Generation

A **Manual** mode is now available alongside the existing **Auto** mode in the Configure Mission panel:

| Mode | Behaviour |
|------|-----------|
| **Auto** | User fills in mission settings, clicks Generate Summary — the API runs the algorithm and renders the output path on the map automatically. |
| **Manual** | User fills in the same settings, clicks **Build & Download Input JSON** to get the planner input file, runs the C++ binary locally, then uploads the output JSON to render the path on the map. No API call is made. |

### Cancel Mission Generation

A **Cancel Generation** button is shown while path generation is in progress. Clicking it stops the background job immediately and resets the form, allowing the user to adjust settings and retry without waiting for the current job to finish or time out.

## Internal Notes

- **Backend** (`metis-demetercloud.com` / `server.js`) — removed v1 code, added `generateMission` support, coverage settings sanitization, planner error detection (422 vs 500), and string return handling from native addon.
- **Frontend** (`app.demetercloud.com`) — obstacle drawing via Google Maps DrawingManager, shared `normalizeMissionOutput` helper for path rendering, Auto/Manual execution mode, `viewFieldContext` SSR fix.
- No database schema changes in this release.
