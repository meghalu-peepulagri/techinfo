# DemeterCloud Admin Portal — Getting Started

## Access

The DemeterCloud Admin Portal is available at:

```
https://admin.demetercloud.peepul.farm
```

## First Login

1. Open the admin portal URL.
2. Enter your credentials (provided by your system administrator).
3. On first login you will be prompted to set a new password and configure 2FA (TOTP).

## Key Concepts

| Concept | Description |
|---------|-------------|
| Organization | Top-level tenant — maps to a company or business unit |
| Farm | A physical farm location within an organization |
| Zone | A subdivision of a farm (field, greenhouse, pump house) |
| Device | An IoT controller (iDhara node, APFC unit, ORC node) assigned to a zone |

## Quick Start Checklist

- [ ] Create your organization
- [ ] Add a farm and zones
- [ ] Provision devices (scan QR code or enter serial number)
- [ ] Invite team members and assign roles
- [ ] Configure alert rules and notification channels
- [ ] Generate an API key for integrations
