# iDhara Admin Dashboard

The iDhara Admin Dashboard is the web interface for fleet managers and system administrators to manage devices, users, and configurations across multiple farms.

## Access

The dashboard is hosted at: `https://admin.demetercloud.peepul.farm`

## Key Sections

| Section | Description |
|---------|-------------|
| Devices | View and manage all field nodes and gateways |
| Farms | Organize devices by farm and zone |
| Users | Manage user accounts and role assignments |
| Alerts | Configure global alert rules and notification channels |
| Reports | Export historical data as CSV |
| Settings | API keys, integrations, and billing |

## Supported Browsers

Chrome 100+, Firefox 100+, Edge 100+, Safari 15+.
