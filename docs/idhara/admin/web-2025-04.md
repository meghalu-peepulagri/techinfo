# iDhara Web Admin — April 2025 Release

**Release Date:**  2026-04-18

---

## What's New

### Device Dispatch Module

- **Add Dispatch Details** — Dispatch details can now be added to a device.
- **View in Device Info** — Dispatch details are displayed within the device info screen.
- **Edit Dispatch Details** — Existing dispatch details can be edited.
- **Expiry Alerts List** — Devices with recharge or warranty expiring within one month are listed for easy tracking.

### Device Versioning

- **Live Version Update** — Device firmware/software version is now updated live from the device info topic (real-time).
