# iDhara

iDhara is Peepul Farm's smart irrigation and soil monitoring IoT platform. It integrates soil sensors, weather data, and automated irrigation control to optimize water usage for agricultural fields.

## Key Components

| Component | Description |
|-----------|-------------|
| Field Node | Edge device with soil moisture, temperature, and EC sensors |
| Gateway | LoRa/4G gateway that relays field node data to DemeterCloud |
| Mobile App | Farmer-facing app for monitoring and manual control |
| Admin Dashboard | Web interface for fleet management and configuration |
