# iDhara Mobile App

The iDhara mobile app is available for **iOS** (14+) and **Android** (10+). It allows farmers and field engineers to monitor sensor data, configure alerts, and trigger irrigation remotely.

## Features

- Real-time sensor dashboard (moisture, temperature, EC)
- Push notifications for threshold alerts
- Manual and scheduled irrigation control
- Historical charts (7-day, 30-day)
- Multi-farm / multi-zone support
- Offline mode with sync on reconnect
