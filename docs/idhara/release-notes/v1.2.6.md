**Dashboard**

---

### Bug Fixes

**Settings & Configuration**
- Dry run limit and overload limit now display their actual values (e.g., seconds/amps) in the confirmation message instead of percentage values
- "Confirm Settings Update" dialog now correctly shows FLC changes alongside voltage and current fault changes when multiple parameters are modified together
- Resolved an intermittent issue where saving configuration required several attempts

**Motor Control & State**
- Motor OFF state now updates immediately without requiring a manual refresh
- Fixed mode change popup incorrectly showing the option to switch to AUTO when the motor was already in AUTO mode
- Fixed an intermittent issue where motor mode displayed as MANUAL instead of AUTO; state now stays consistent without needing a refresh

**Test Run**
- Test run timer now updates correctly even when the app is running in the background
- Added retry logic for motor ON/OFF commands at the start and end of a test run (retry interval: 5 seconds, retry count: 2) to prevent the motor from staying ON if the OFF command is missed

**Sync**
- Fixed previous and current values appearing identical in the settings change confirmation
- Improved real-time sync between the mobile app and the web admin panel

**Notifications & Fault Display**
- Fixed incorrect "Power Failure" notification being shown when the motor turns off in AUTO mode without an actual power failure
- Fixed "After Power Recovery" message appearing every time the motor is set to AUTO mode even when no power failure had occurred
- Fixed "No More Faults" notification appearing multiple times when an active fault (e.g., input phase failure) is present
- Fixed duplicate notification messages when the content was identical

---

### Device Info

- Added the following fields to the device info section: Starter Number, PCB Number, Test Run Date, Device Location, and Device Image

### Weather

- Weather scroll bar now defaults to showing the current time ("Now") when the app is opened instead of the start of the day
