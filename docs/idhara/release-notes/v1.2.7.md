### Bug Fixes

**Settings & Configuration** *(Bug #58)*
- Dry run limit and overload limit now display their actual configured values (e.g., seconds, amps) in the confirmation message instead of percentage values
- "Confirm Settings Update" dialog now correctly shows FLC changes alongside voltage and current fault changes when multiple parameters are modified together

**Test Run** *(Bug #64)*
- Test run timer now decrements correctly when the app is running in the background or closed
- Added retry logic for motor ON/OFF commands at the start and end of a test run to prevent the motor from remaining ON if the OFF command is missed (retry interval: 5 seconds, retry count: 2)

**Device Connectivity** *(Bug #78)*
- Fixed "Device not connected" status incorrectly shown even when the device is actively responding to commands
- Network connectivity status now updates in real time
- Sequential connectivity ticks now display in the correct chronological order
