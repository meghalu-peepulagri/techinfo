# iDhara v1.2.4 — Internal Release

**Release Date:** 2026-04-12

> **Superseded by [v1.2.5](./v1.2.5)** — released 2026-04-20

---

## Mobile App (v1.2.4)

📦 **Download:** [iDhara-prod-v1.2.4.apk](pathname:///releases/iDhara-prod-v1.2.4.apk)  
*(Play Store link will replace this once published)*

**Motor & Fault Management**
- Updated the Motor Settings screen title for clarity
- Added a "Clear Fault" option when the motor is turned on
- Users can enable or disable individual faults
- Fault settings screen redesigned to show on/off status and details per fault

**Dashboard**
- Device runtime now shown in the device card footer
- Fault status displayed on each device card
- Faults can be cleared directly from the dashboard
- Last motor activity state is displayed on the dashboard

**Informational**
- Added information about Auto and Manual modes
- Added details about FLC, overload, dry run, and fault settings
- Warning shown when motor is temporarily off: *"This is temporarily turned off. The motor will turn on after a power recycle."*

**UX & Navigation**
- Settings moved to a separate menu item
- Device image can be captured while adding a new device
- Separate builds maintained for Development and Production environments

---

## Web Admin

**Device Dispatch Module**
- Dispatch details can be added, viewed, and edited on a device
- Devices with recharge or warranty expiring within one month are listed for easy tracking

**Device Versioning**
- Device version updates live from the device info topic in real-time
