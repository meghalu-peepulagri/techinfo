# ORC — Organic Resource Controller

ORC is Peepul Farm's IoT device for monitoring and controlling organic resource inputs (fertigation, composting, biogas) on the farm. It provides sensor feedback and automated dosing control.

## Key Components

| Component | Description |
|-----------|-------------|
| ORC Node | Edge controller with pH, EC, flow, and level sensors |
| Actuator Module | Relay/motor driver for pump and valve control |
| DemeterCloud Integration | Remote monitoring and scheduling |
