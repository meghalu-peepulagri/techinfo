# ORC — Getting Started

## Prerequisites

- ORC Node unit
- pH and EC probes (calibrated)
- Flow sensor (pulse output)
- DemeterCloud account

## Installation Steps

1. **Mount** the ORC node near the fertigation tank or dosing point.
2. **Connect sensors** — pH probe to port A, EC probe to port B, flow sensor to digital input 1.
3. **Connect actuators** — dosing pumps to relay outputs (Relay 1–4).
4. **Power on** — the node runs a sensor calibration check on first boot.
5. **Register in DemeterCloud** — scan the QR code on the unit to auto-provision.
6. **Create a dosing schedule** in DemeterCloud under _ORC → Schedules_.
